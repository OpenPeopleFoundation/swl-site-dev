# Deep-Space Design System — v1.0

## Purpose
Provide a unified visual and interaction language for all AI-facing UI in the Open People ecosystem. Interfaces must evoke:
- gravitational depth
- event-horizon curvature
- holographic data surfaces
- minimalism within negative space
- AI-first usability with human-legibility

---

## Core Aesthetic Keywords
- Neural glass
- Black-hole depth
- Photon-surface shimmer
- Silent void gradients
- Floating panes
- Event horizon rings
- Low-orbit motion
- Lightfield overlays

---

## Color System

### Core Void
- `#000000` — Absolute black  
- `#020205` — Void blue-black  
- `#0C0F17` — Subsurface graphite  

### Holographic Neons
- `#00F3FF` — Cyan lens-flare  
- `#9B4DFF` — Singularity violet  
- `#00FF9C` — Life-vector green  
- `#FF8EFF` — Nebula pulse  

### Metallics
- `#C0C0C0` — Chrome  
- `#616161` — Gunmetal  

---

## Typographic Language
**Primary:** Eurostile Next / Square grotesk  
**Secondary:** Inter (system readability)

- Large titles: condensed, tight leading  
- Body: soft, generous letter spacing  
- Captions: mono-hinting for “technical sheet” feel  

---

## Motion
Use motion to imply:
- curvature of space  
- slow orbital drift  
- holographic panel activation  

Rules:
- No jitter  
- No bouncing  
- No spring physics  
- Slow, deliberate, slightly cinematic  

---

## Components
### Holographic Panel
- Floating glass with 8–12% translucency
- 1px photon rim (cyan or violet)
- Subtle interior bloom

### Command Input Window
- Ultra-minimal  
- Floating matte black capsule  
- 1px neon rim when active  
- Slight inward warp on focus  

---

## Layout Language
- Everything floats  
- No hard blocks stuck to viewport  
- Panels orbit around a central gravitational anchor  
- Use Z-layer depth aggressively
